# CAD

CAD files are not yet included in this repository.

This folder will hold design files, printable prototypes, exported models, and notes for mechanical assemblies.

## CAD Sections

### Turret System

Design files for the rotating turret assembly. This section replaces the earlier
`turret_worm_drive` area while keeping the worm-drive turret concept as part of
the full turret system.

Folder: `turret_worm_drive/`

### Mower Subsystem

Parts and assemblies for the mower mechanism, including blade mounting,
protective structure, lift parts, and mower-to-frame interfaces.

Folder: `mower_subsystem/`

### Drivetrain

Models for the wheel, motor, axle, gearbox, and drive mounting components used
to move the rover.

Folder: `drivetrain/`

### Outer Frame

Structural frame pieces, brackets, mounting rails, and exterior support parts
that define the robot chassis.

Folder: `outer_frame/`

### Electronics Holder

Mounts and trays for general electronics, wiring support, sensors, power
distribution, and controller boards.

Folder: `electronics_holder/`

### ODrive Holder

Dedicated CAD for mounting and protecting the ODrive motor controller hardware.

Folder: `odrive_holder/`

### Computer Holder

Mounting parts for the onboard computer, including support plates, standoffs,
and airflow or cable-management features.

Folder: `computer_holder/`

### Full Robot

Top-level CAD assembly for the complete autonomous garden management rover,
combining the turret system, mower subsystem, drivetrain, frame, and electronics
mounting.

Folder: `full_robot/`

The current README image uses a CAD overview render in `media/cad_overview.jpg`.
