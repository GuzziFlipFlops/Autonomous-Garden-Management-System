# Turret Worm Drive

The turret base uses a printable worm-drive prototype.

The current design target is a 52-tooth module-4 worm wheel with a 1-start worm for a 52:1 ratio.

Printable worm gears should be physically tested before use because printed gear geometry and backlash depend on material, printer calibration, layer orientation, and printer tolerances.

## Current Role

This mechanism is the first degree of freedom for the rear hose turret. It rotates the hose aiming base. A second degree of freedom for hose up/down angle will be built later.

## Testing Needed

- Backlash measurement
- Gear wear checks
- Rotation repeatability
- Load testing with hose weight
- Limit and stall behavior
