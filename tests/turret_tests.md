# Turret Tests

## Worm-Drive Base

- Check gear mesh.
- Measure backlash.
- Rotate through the expected range.
- Check for binding.
- Test repeatability.
- Inspect printed gear wear.

## Hose Load

- Test rotation with hose weight attached.
- Check if hose drag affects aiming.
- Check motor and gear temperatures.

## Water Tests

- Test with water first.
- Verify flow control.
- Verify hose aiming.
- Verify safe shutoff.
- Keep electrical connections protected from water.
