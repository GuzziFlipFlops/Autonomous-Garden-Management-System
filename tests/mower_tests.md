# Mower Tests

## Pre-Power Checks

- Confirm mower hardware is secure.
- Confirm guards or safe test setup are in place.
- Confirm emergency stop or main disconnect works.
- Keep people and pets away from the test area.

## Controller Checks

- Test the low-cost ODrive-style FOC controller at low power.
- Verify safe enable/disable behavior.
- Monitor temperature and current.
- Stop testing if vibration, heat, or wiring issues appear.

## Cutting Tests

Cutting tests should only happen after drivetrain, lift, and mower control are reliable. Start with conservative settings and controlled conditions.
