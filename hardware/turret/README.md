# Turret Hardware

The rear turret is being developed as a 2-DOF hose aiming module.

## Current Status

Phase 1 is complete or in progress: a worm-gear rotating base for the first degree of freedom.

The second degree of freedom for hose up/down angle will be built later.

## Base Station

The robot will connect to a base station with a hose attachment. The hose can be autonomously attached or detached when the robot is parked at the base station.

## Notes

The turret can aim water toward plants. Any controlled liquid application module should be tested with water first.
