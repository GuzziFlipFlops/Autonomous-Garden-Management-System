# Control Box Hardware

The control box is still in progress.

## Planned Hardware

- ESP32-P4 high-level controller
- Two STM32 F103 Blue Pill boards
- Motor driver wiring
- Power distribution
- Signal isolation/protection
- Sensor terminals
- Main disconnect and safety wiring
- Debug/programming access

## Design Notes

The box should make wiring serviceable and safe to inspect. High-current paths, low-voltage logic, and sensor wiring should be separated and labeled where practical.
