# Turret Module

The rear turret is being developed as a 2-DOF hose aiming module for plant-targeted watering.

## Current Status

Phase 1 is complete or in progress: a worm-gear rotating base for the first turret degree of freedom.

The second turret degree of freedom will be built later and will control hose up/down angle.

## Hose Docking

The robot will connect to a base station with a hose attachment. When parked at the base station, the hose can be autonomously attached or detached.

## Planned Capabilities

- Rotate the hose turret toward plant areas
- Adjust hose elevation with a second axis
- Control water flow from the base-station hose system
- Support modular rear garden tooling
- Optionally support controlled liquid application with water-first testing

## Testing Needed

- Worm-drive backlash and repeatability checks
- Printed gear wear checks
- Hose load testing
- Rotation limit testing
- Base-station attach/detach testing
- Water-only flow testing
