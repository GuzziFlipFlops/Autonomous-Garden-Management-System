# Lift Mechanism

The mower up/down lift mechanism is mechanically done. It uses a custom rope-based lift system.

## Purpose

The lift raises and lowers the mower module so cutting height can be adjusted electronically. This avoids a heavier or more expensive actuator approach while keeping the mechanism serviceable during early development.

## Current Status

- Mechanical lift path: done
- Rope-based system: done
- Repeated-load testing: still needed
- Position repeatability testing: still needed
- Firmware integration: still needed

## Notes

The lift should be tested carefully before mower-enabled operation. Rope stretch, knot security, pulley friction, and frame flex can all change the final mower position.
