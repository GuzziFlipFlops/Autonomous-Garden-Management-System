# ESP32-P4 High-Level Controller

This folder will contain the ESP32-P4 firmware for high-level robot coordination.

## Planned Responsibilities

- Designed autonomous operation logic
- Subsystem state management
- Communication with STM32 low-level controllers
- Mower enable coordination
- Lift, drivetrain, turret, and docking behavior coordination
- Test logging where practical

## Status

Firmware is in progress and has not been added to this repository yet.
