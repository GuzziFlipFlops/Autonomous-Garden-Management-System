# STM32 Low-Level B

This folder will contain firmware for the second STM32 F103 Blue Pill low-level controller.

## Planned Responsibilities

- Additional motor or actuator I/O
- Mower lift control
- Turret I/O
- Sensor inputs not assigned to the first STM32 board

## Status

Firmware is in progress and has not been added to this repository yet. Responsibilities may change as wiring is finalized.
