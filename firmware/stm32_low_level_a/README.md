# STM32 Low-Level A

This folder will contain firmware for one STM32 F103 Blue Pill low-level controller.

## Planned Responsibilities

- Drivetrain encoder reading
- Motor PWM generation
- BTS7960 control
- Low-level drivetrain sensor I/O

## Status

Firmware is in progress and has not been added to this repository yet. The exact pin map is still being finalized.
